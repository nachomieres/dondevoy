dondeVoy
=====================

## Uso del proyecto:

You'll need the Ionic CLI with support for v2 apps:

```bash
$ npm install -g ionic
```

Then run:

```bash
$ ionic start dondevoy https://github.com/nachomieres/dondevoy --v2
```
